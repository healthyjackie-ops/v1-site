Business Model

三条收入线，覆盖全生命周期

从 IP 授权到视觉接入芯片，再到软件与数据服务——
V1 构建的是一个从一次性收入到持续性收入的完整商业闭环。

IP
IP 授权
IP Licensing

将 V1 视觉处理 pipeline 作为 IP 核授权给 SoC 厂商集成， 类似 ARM / VeriSilicon 模式，快速触达多家芯片平台。

收入模式 授权费 + 版税
毛利率 ~90%+
启动速度 最快
芯
视觉接入芯片
Vision ASIC

自研轻量级视觉接入 ASIC——含 MIPI RX/TX、ISP 等视觉处理模块，无 DDR，可选 RISC-V 内核。类似相机模组中集成的 ISP 芯片，成本极低，可反复投片快速迭代。

收入模式 芯片一次性销售
架构 专用视觉前端，非通用 SoC
成本优势 极低 BOM，可反复投片
壁垒 ASIC 硬化 + 闭源模型
软
软件 & 数据服务
Software & Data Services

覆盖视觉数据全生命周期的工具链与服务， 按项目或按量持续付费，高毛利，远期价值远超芯片。
